# 🔧 BorrowBuddy – Item Lending App

A full-stack web app to borrow and lend items in your neighborhood.

---

## 🚀 Features

- User Registration & Login  
- Add & Browse Items  
- Borrow Requests with Owner Approval  
- Reviews & Ratings  
- Session Timeout & Protected Routes  
- Responsive Design

---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Spring Boot (Java)  
- **Database:** MySQL  
- **Tools:** Axios, Postman, Git, Workbench

---

## 📁 Project Structure

```
BorrowBuddy/
├── backend/     # Spring Boot project
├── frontend/    # React app
├── db/          # MySQL .sql file
└── README.md
```

---

## 💻 Setup Guide

### Backend:
- Run Spring Boot app in Eclipse
- API base URL: `http://localhost:8080`

### Frontend:
```bash
cd frontend
npm install
npm start
```

### Database:
- Import `db/borrowbuddy.sql` into MySQL Workbench
